1. ...です
   解说： 表示判定，翻译为 ...是...
   接续：名词 + は／が　+ 名词 + です
   例句：たなかさん　は　がくせい　です　　田中是学生
        ちちはせんせいです　　　　　　　　父亲是老师
2. ...ではありません　　
   解说：表示判定，翻译为 ...不是...
   口语中常用语 ...じゃありません
   接续：名词 + は／が　＋　ではありません
   例句：たなかさんはがくせいではありません　田中先生不是学生
        ちちはせんせいじゃありません　父亲不是老师
3. ...も
   解说：翻译为 ...也....
   接续：名词　+　も
   例句：みせでほんをかいました。ペンも買いました　在商店买了书，也买了笔
        ビールをのみました。　ウインも飲みました　我喝啤酒，也喝红酒
4. ...で
    解说：表示场所，翻译为 在...
    接续：场所名词 + で
    例句：しょくどう　で　ごはん　を　たべます　在食堂吃饭
         わたしのいえ　で　遊ぼう　在我家玩吧
5. ...に
   解说：表示移动到达的场所
   接续：目的地名词 + に
   例句：とうきょう　に　行きます　去东京
        いえ　に　帰ります　回家
6. ...に
    解说：表示事情发生的时间点
    接续：时间点 + に
    例句：しちじ　に　いえ　を　出ます　　7点出门
         こどし　の　さんがつ　に　にほん　に　来ました　今年3月来日本了
7. ...を
    解说：表示他动词的作用对象
    接续：名词 + を　+他动词
    例句：ごはんを　食べます　吃饭
        　にほんご　を　勉強します　学习日语
8. ...後で　（あとで）
    解说：表示 ...之后...
    接续：名词 + の + あとで
        　动词た行 + あとで
    例句：しごと　の　あとで、　のみ　に　行きませんか
         下班后，要不要去喝一杯？
         えいが　を　見た後で、　映画の名前の意味がわかりました
         看完电影后，我明白了电影名字的意义 （見る的た形 みた）
9.  ...がいちばん...
    解说：表示三个以上事物做比较，程度最高的那个翻译为 最...
    接续：N + がいちばん
    日本利用理の中で、寿司が一番好きです
    日本料理中，我最喜欢寿司了
    くるまの　なかで、　トヨタがいちばん　すきです
    车子中，我最喜欢丰田
10. ...がほしい　が欲しい
    解说：表示说话人的愿望，希望。翻译为 想要...
    接续：名词 + が欲しい
    例句：彼女（かのじょ）　が　ほしい　です
         我想要女朋友
         たんじょうび　に　なに　がほしい　ですが
         生日你想要什么？
11. ...から
    解说：表示理由，原因 翻译为 因为...
    接续: 文1 + から + 文2
    例句：ねむい　から　ねます　　因为困所以睡觉
         やさい　が　きらいです　から、　たべたくないです
         野菜が　嫌いです　から、　食べたくないです
         因为讨厌蔬菜，所以不想吃
12. あまり...ない
    解说：表示程度不那么高，翻译为 不太.../ 没怎么
    接续：あまり　+ 否定形
    例句：きのう　は　あまり　べんきょうしません　でした
        　昨日　は　あまり　勉強　（过去否定形）しません　でした
        　昨天没怎么学习
          おさけ　は　あまり　すきじゃありません
          お酒　は　あまり　好きじゃ　ありません
          不怎么喜欢喝酒
13. ...だけ
    解说：翻译为 仅仅... 只....
    接续：N + だけ
    例句：さいふ　の　なか　に　にひゃくえん　だけ　あります
         財布　の　中　に　二百円　だけ　あります
         钱包里只有200日元
         わたし　の　やすみ　は　にちようび　だけ　です
         私　の　休み　は　日曜日　だけ　です
         我的休息日只有周日
14. ...で
    解说：表示手段，方法。翻译为 用...
    接续：N + で
    例句：おはし　で　ごはんをたべます
         お箸　で　ご飯　を　たべます
         用筷子吃饭
         スマホ　で　しゃしん　を　とります
         スマホ　で　写真　を　撮ります
         用手机拍照
15. ...てから
    解说：翻译为 ...之后在..., 然后...
    接续：动词て形 + から
    例句：テレビ　を　みてから、　べんきょうします
    　　　テレビ　を　見てから、　勉強します　（見る）
         看电视之后在学习
         て　を　あらつ　てから　ごはんをたべます
         手　を　洗ってから　ごはんをたべます （洗う）
         洗手之后在吃饭
16. とても
    解说：表示程度很高，翻译为 非常...
    口语中常用 とっても
    例句：きょう　は　とても　あついですね
         今日は　とても　暑いですね
         今天很热啊
         こ　の　ほん　は　とても　おもしる　いです
         この本は　とても　面白い　です
         这本书很有趣
17. ...でじょう
    解说：表示向对方确认，或者征求对方同意 翻译为 ...吧？
    例句：これ、きのう　かれし　に　もらってんだ。　いいでしょう？
         これ、昨日　彼氏　に　もらったんだ。　いいでしょう？
         这个是昨天男朋友送我的，很棒吧？
         ええ、いいなあ
         嗯，真好啊
18. ...とおもう　...と思う
　　 解说：表示陈述意见或者推测 翻译为 （我）认为... （我）觉得..
    接续：普通形 + と思う
    例句：たべすぎ　は　からだ　に　よくない　とおもいます
        　食べ過ぎ（吃太多）　は　体　に　よくない（不好）　と思います。
        　我觉得吃太多对身体不好
         にほんの　ぶっか　は　たかい　とおもい　ますか
         日本の　物価　は　高い　と思い　ますか
         你觉得日本的物价高吗？
19. ...たい
    解说：翻译为 想要... 想...
    接续：V（ます形）删掉ます　+ たい
    例句：こんや　は　ラーメン　か　たべたいです
         今夜　は　ラーメン　が　食べたいです （食べる）
         今晚想吃拉面
         きょう　は　いえ　で　やすみたい　です
         今日　は　家　で　休みたい　です　（休む）
         今天想在家里休息
20. ...とき
　　解说：翻译为 ...的时候
    接续：V（普）+ とき
        イAい　+ とき
        ナAな　+ とき
        N + のとき
    例句：じかん　が　ない　とき　は、　あさはん　を　たべません
        　時間が　ないときは、　朝ごはんを　食べません
         没有时间的时候， 就不去吃早饭
         ねるとき、　わたし　は　でんきをけしません
         寝るとき、　私　は　電気を消しません
         睡觉的时候，我不关灯
21. ...てもいいですか
　　解说：表示征求同意，寻求许可，翻译为 我可以...吗？
   接续：V（て形）+ もいいですか
   例句：ここに　すわって　もいいですか
        ここに　座って　もいいですか （座る）
        我可以做这里吗？
        タバコ　を　すって　もいいですか
        タバコ　を　吸って　もいいですか　（吸う）
        我可以吸烟吗？
