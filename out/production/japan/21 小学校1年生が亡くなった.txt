【小学校1年生が亡くなった】
福岡県みやま市の小学校で26日、1年生の男の子が給食の事故で亡くなりました。  
うずらの卵がのどに詰まったことが原因のようです。うずらの卵は直径2cmぐらいで、切らないで給食に入っていました。  
みやま市はうずらの卵を給食でしばらくの間使わないと言いました。  
国は、給食に気をつけるように日本中の学校に言いました。子どもたちには、よくかんでゆっくり食べるように伝えます。
先生には、子どもたちが食べている間、気をつけて見ているように伝えます。


 句子1
福岡県みやま市の小学校で26日、1年生の男の子が給食の事故で亡くなりました。
 读法：ふくおかけん みやまし の しょうがっこう で 26にち、いちねんせい の おとこのこ が きゅうしょく の じこ で なくなりました。
 翻译：福冈县宫山市的小学在26日，一年级的男孩因午餐事故去世了。
 语句分析：
   福岡県（ふくおかけん）：名词，地点，福冈县。
   みやま市（みやまし）：名词，地点，宫山市。
   の：连接助词，表示所属关系。
   小学校（しょうがっこう）：名词，小学，主语。
   で：地点标记助词，表示发生的地点。
   26日（26にち）：名词，表示时间。
   、：顿号，用于分隔成分。
   1年生（いちねんせい）：名词，一年级学生，主语补充成分。
   の：连接助词，表示所属。
   男の子（おとこのこ）：名词，男孩，主语。
   が：主语标记助词。
   給食（きゅうしょく）：名词，午餐，表原因。
   の：连接助词，表示所属。
   事故（じこ）：名词，事故，表原因。
   で：原因标记助词。
   亡くなりました（なくなりました）：动词，去世的过去式。



 句子2
うずらの卵がのどに詰まったことが原因のようです。
 读法：うずらのたまご が のど に つまった こと が げんいん の ようです。
 翻译：似乎是鹌鹑蛋卡在喉咙里导致的。
 语句分析：
   うずらの卵（うずらのたまご）：名词，鹌鹑蛋，主语。
   が：主语标记助词。
   のど（のど）：名词，喉咙，表位置。
   に：方向标记助词。
   詰まった（つまった）：动词，堵住了，过去式。
   こと：名词，事情，抽象表达。
   が：主语标记助词。
   原因（げんいん）：名词，原因。
   の：连接助词，表示所属。
   ようです：表示推测的表达方式。



 句子3
うずらの卵は直径2cmぐらいで、切らないで給食に入っていました。
 读法：うずらのたまご は ちょっけい 2センチメートル ぐらい で、きらないで きゅうしょく に はいっていました。
 翻译：鹌鹑蛋的直径大约为2厘米，没有切开就放入午餐中。
 语句分析：
   うずらの卵（うずらのたまご）：名词，鹌鹑蛋，主语。
   は：主题标记助词。
   直径（ちょっけい）：名词，直径，表描述。
   2cm（2センチメートル）：数量词，2厘米。
   ぐらい：表示大约的数量。
   で：状态标记助词。
   、：顿号，用于分隔部分。
   切らないで（きらないで）：动词，表示不切。
   給食（きゅうしょく）：名词，午餐，表位置。
   に：方向标记助词。
   入っていました（はいっていました）：动词，放入的过去式。



 句子4
みやま市はうずらの卵を給食でしばらくの間使わないと言いました。
 读法：みやまし は うずらのたまご を きゅうしょく で しばらくのあいだ つかわない と いいました。
 翻译：宫山市表示将在一段时间内不再在午餐中使用鹌鹑蛋。
 语句分析：
   みやま市（みやまし）：名词，宫山市，主语。
   は：主题标记助词。
   うずらの卵（うずらのたまご）：名词，鹌鹑蛋，宾语。
   を：宾语标记助词。
   給食（きゅうしょく）：名词，午餐，表方式。
   で：方式标记助词。
   しばらくの間（しばらくのあいだ）：名词，一段时间。
   使わない（つかわない）：动词，表示不使用。
   と：引述助词，引导间接引语。
   言いました（いいました）：动词，表示说的过去式。



 句子5
国は、給食に気をつけるように日本中の学校に言いました。
 读法：くには、きゅうしょく に きを つける よう に にほんじゅう の がっこう に いいました。
 翻译：国家对全国学校表示，要注意午餐安全。
 语句分析：
   国（くに）：名词，国家，主语。
   は：主题标记助词。
   、：顿号，用于分隔部分。
   給食（きゅうしょく）：名词，午餐，表对象。
   に：方向标记助词。
   気をつける（きをつける）：动词，表示注意。
   ように：表示目的或方式的助词。
   日本中の学校（にほんじゅうのがっこう）：名词，全国学校，宾语。
   に：方向标记助词。
   言いました（いいました）：动词，表示说的过去式。



 句子6
子どもたちには、よくかんでゆっくり食べるように伝えます。
 读法：こどもたちには、よく かんで ゆっくり たべる よう に つたえます。
 翻译：将告诉孩子们要认真咀嚼，慢慢吃。
 语句分析：
   子どもたち（こどもたち）：名词，孩子们，主语。
   には：方向标记助词，表示"对……"。
   、：顿号，用于分隔部分。
   よく：副词，表示好好地。
   かんで（かむ）：动词，咀嚼的连用形。
   ゆっくり：副词，慢慢地。
   食べる（たべる）：动词，吃，基态。
   ように：表示目的或方式的助词。
   伝えます（つたえます）：动词，传达的现在式。
