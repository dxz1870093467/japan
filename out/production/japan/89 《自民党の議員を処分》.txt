《自民党の議員を処分》
自民党のグループがパーティーで集めたお金を正しく報告していなかったことがわかって、問題になっています。
自民党は4日、この問題について話し合いました。そして、39人の議員を処分することを決めました。
グループの中心になっていた塩谷議員と世耕議員の2人には、自民党から出るように言いました。
8つの処分の中で、2番目に厳しい処分です。
このため世耕議員は、4日の夜に自民党から出ました。
37人については、3番目から7番目に厳しい処分にしました。
自民党は、同じことがまた起こらないように、法律を変えることなどを考えると言っています。
立憲民主党などの野党は「問題が起こった原因が十分にわかっていません。
しっかり説明するように、これからも自民党に強く言っていきます」と話しています。


自民党のグループがパーティーで集めたお金を正しく報告していなかったことがわかって、問題になっています。

 读法：じみんとう の グループ が パーティー で あつめた おかね を ただしく ほうこくしていなかった こと が わかって、もんだい に なっています。
 翻译：自民党的一个团体没有正确报告在派对上筹集的钱，这件事被发现后成为了问题。
 语法分析：

   「〜て、問題になっています」表示“因为……而成为问题”。
   「〜ことがわかって」表示“发现了……的事实”。
   「報告していなかった」是「報告する」的否定过去进行时，表示“没有报告”。



自民党は4日、この問題について話し合いました。

 读法：じみんとう は よっか、この もんだい について はなしあいました。
 翻译：自民党4日就这件问题进行了讨论。
 语法分析：

   「〜について」表示“关于……”。
   「話し合いました」表示“进行了讨论”。



そして、39人の議員を処分することを決めました。

 读法：そして、さんじゅうきゅうにん の ぎいん を しょぶんする こと を きめました。
 翻译：然后决定处分39名议员。
 语法分析：

   「〜ことを決めました」表示“决定做……”。
   「処分する」表示“处分、处理”。



グループの中心になっていた塩谷議員と世耕議員の2人には、自民党から出るように言いました。

 读法：グループ の ちゅうしん に なっていた しおたに ぎいん と せこう ぎいん の ふたり に は、じみんとう から でる ように いいました。
 翻译：成为团体核心的盐谷议员和世耕议员两人被要求退出自民党。
 语法分析：

   「〜ように言いました」表示“要求做……”。
   「〜から出る」这里指“退出（党）”。



8つの処分の中で、2番目に厳しい処分です。

 读法：やっつ の しょぶん の なか で、にばんめ に きびしい しょぶん です。
 翻译：这是8种处分中第二严重的处分。
 语法分析：

   「〜の中で」表示“在……之中”。
   「〜番目に」表示“第……位”。
   「厳しい」表示“严厉的”。



このため世耕議員は、4日の夜に自民党から出ました。

 读法：このため せこう ぎいん は、よっか の よる に じみんとう から でました。
 翻译：因此，世耕议员在4日晚上退出了自民党。
 语法分析：

   「このため」表示“因此”。
   「〜から出ました」表示“退出了（某组织）”。



37人については、3番目から7番目に厳しい処分にしました。

 读法：さんじゅうななにん について は、さんばんめ から ななばんめ に きびしい しょぶん に しました。
 翻译：对于37人，给予了第三到第七等级的处分。
 语法分析：

   「〜については」表示“关于……”。
   「〜にしました」表示“决定为……”。



自民党は、同じことがまた起こらないように、法律を変えることなどを考えると言っています。

 读法：じみんとう は、おなじ こと が また おこらない ように、ほうりつ を かえる こと など を かんがえる と いっています。
 翻译：自民党表示会考虑修改法律等措施，以防止同样的事情再次发生。
 语法分析：

   「〜ように」表示“为了……”。
   「〜と言っています」表示“说，表示”。
   「ことなど」表示“……等事情”。



立憲民主党などの野党は「問題が起こった原因が十分にわかっていません。

 读法：りっけんみんしゅとう など の やとう は 「もんだい が おこった げんいん が じゅうぶん に わかっていません。
 翻译：立宪民主党等在野党表示，“问题发生的原因还没有被充分弄清楚”。
 语法分析：

   「〜など」表示“等等”。
   「〜ていません」否定进行时，表示“还没有……”。



しっかり説明するように、これからも自民党に強く言っていきます」と話しています。

 读法：しっかり せつめい する ように、これからも じみんとう に つよく いっていきます」 と はなしています。
 翻译：“会继续强烈要求自民党进行充分的说明。”（他们）说。
 语法分析：

   「〜ように」表示目的。
   「〜ていきます」表示动作持续进行或将来的动作。
