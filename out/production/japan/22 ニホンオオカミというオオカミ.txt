【ニホンオオカミというオオカミ】
日本には、ニホンオオカミというオオカミがいましたが、100年ぐらい前にいなくなったと言われています。はく製や標本はあまり残っていません。  
国立科学博物館にあるはく製の動物がニホンオオカミの可能性が高いとわかりました。今月、中学校1年生の小森日菜子さんと専門家が発表しました。  
小森さんは、小さいときからニホンオオカミが好きで、勉強していました。小学生のとき、茨城県の国立科学博物館の倉庫にあるはく製を見て、「ニホンオオカミに似ている」と考えました。  
専門家と一緒に研究すると、このはく製は100年以上前に上野動物園で飼っていたニホンオオカミの可能性が高いことがわかりました。  
小森さんは「顔や足の形などを見て、これはニホンオオカミだとすぐに感じました。うれしくて、踊りたくなりました」と話していました。


 句子1
日本には、ニホンオオカミというオオカミがいましたが、100年ぐらい前にいなくなったと言われています。
 读法：にほん には、にほんおおかみ という おおかみ が いました が、100ねんぐらい まえ に いなくなった と いわれています。
 翻译：在日本曾经有一种叫做“日本狼”的狼，但据说大约在100年前就消失了。
 语句分析：
   日本には（にほんには）：名词，表示地点，在日本。
   ニホンオオカミというオオカミがいましたが（にほんおおかみというおおかみがいましたが）：名词，存在的狼。
   100年ぐらい前にいなくなったと言われています（100ねんぐらいまえにいなくなったといわれています）：动词，表示消失的过去式，引用的说法。



 句子2
はく製や標本はあまり残っていません。
 读法：はくせい や ひょうほん は あまり のこっていません。
 翻译：标本和标本并没有留下太多。
 语句分析：
   はく製や標本は（はくせいやひょうほんは）：名词，标本和标本，主语。
   あまり残っていません（あまりのこっていません）：动词，表示没有留下的现在式。



 句子3
国立科学博物館にあるはく製の動物がニホンオオカミの可能性が高いとわかりました。
 读法：こくりつかがくはくぶつかん に ある はくせい の どうぶつ が にほんおおかみ の かのうせい が たかい と わかりました。
 翻译：在国立科学博物馆的标本动物被认为有很高的可能性是日本狼。
 语句分析：
   国立科学博物館にある（こくりつかがくはくぶつかんにある）：名词，位于国立科学博物馆的。
   はく製の動物が（はくせいのどうぶつが）：名词，标本动物，主语。
   ニホンオオカミの可能性が高いとわかりました（にほんおおかみのかのうせいがたかいとわかりました）：动词，得知有很高的可能性是日本狼。



 句子4
今月、中学校1年生の小森日菜子さんと専門家が発表しました。
 读法：こんげつ、ちゅうがっこう 1ねんせい の こもり ひなこ さん と せんもんか が はっぴょうしました。
 翻译：这个月，中学一年级的学生小森日菜子和专家进行了发表。
 语句分析：
   今月（こんげつ）：名词，这个月，表示时间。
   中学校1年生の（ちゅうがっこう1ねんせいの）：名词短语，中学一年级的。
   小森日菜子さんと専門家が（こもりひなこさんとせんもんかが）：名词，小森日菜子和专家，主语。
   発表しました（はっぴょうしました）：动词，进行了发表的过去式。



 句子5
小森さんは、小さいときからニホンオオカミが好きで、勉強していました。
 读法：こもりさんは、ちいさい とき から にほんおおかみ が すき で、べんきょう していました。
 翻译：小森小姐从小就喜欢日本狼，并且一直在学习。
 语句分析：
   小森さんは（こもりさんは）：名词，小森小姐，主语。
   小さいときから（ちいさいときから）：副词，从小的时候开始。
   ニホンオオカミが好きで（にほんおおかみがすきで）：动词，喜欢日本狼。
   勉強していました（べんきょうしていました）：动词，过去式，一直在学习。



 句子6
小学生のとき、茨城県の国立科学博物館の倉庫にあるはく製を見て、「ニホンオオカミに似ている」と考えました。
 读法：しょうがくせい の とき、いばらきけん の こくりつかがくはくぶつかん の そうこ に ある はくせい を みて、「にほんおおかみに にている」と かんがえました。
 翻译：在小学时，看到茨城县国立科学博物馆仓库里的标本，考虑到“与日本狼相似”。
 语句分析：
   小学生のとき（しょうがくせいのとき）：名词，指小学时。
   茨城県の国立科学博物館の倉庫にある（いばらきけんのこくりつかがくはくぶつかんのそうこにある）：名词短语，位于茨城县国立科学博物馆仓库的。
   はく製を見て（はくせいをみて）：动词，看到标本。
   「ニホンオオカミに似ている」と考えました（「にほんおおかみににている」とかんがえました）：引述，认为与日本狼相似的过去式。



 句子7
専門家と一緒に研究すると、このはく製は100年以上前に上野動物園で飼っていたニホンオオカミの可能性が高いことがわかりました。
 读法：せんもんか と いっしょに けんきゅう すると、この はくせい は 100ねん いじょう まえ に うえのどうぶつえん で かっていた にほんおおかみ の かのうせい が たかい ことが わかりました。
 翻译：与专家一起研究后，发现这个标本很可能是100多年前在上野动物园饲养的日本狼。
 语句分析：
   専門家と一緒に研究すると（せんもんかといっしょにけんきゅうすると）：动词，进行研究。
   このはく製は100年以上前に（このはくせいは100ねんいじょうまえに）：名词短语，这个标本是100多年前。
   上野動物園で飼っていたニホンオオカミの可能性が高いことがわかりました（うえのどうぶつえんでかっていたにほんおおかみのかのうせいがたかいことがわかりました）：动词，得知很可能是上野动物园饲养的日本狼。

 句子8（续）
 小森さんは「顔や足の形などを見て、これはニホンオオカミだとすぐに感じました。うれしくて、踊りたくなりました」と話していました。  
 读法：こもりさんは「かお や あし の かたち など を みて、これは にほんおおかみ だ と すぐ に かんじました。うれしくて、おどりたくなりました」と はなしていました。
 翻译：小森小姐说：“看到脸和脚的形状后，我立刻感到这是日本狼。我感到非常高兴，想要跳舞。”
 语句分析：
   小森さんは（こもりさんは）：名词，小森小姐，主语。
   「顔や足の形などを見て、これはニホンオオカミだとすぐに感じました」（「かおやあしのかたちなどをみて、これはにほんおおかみだとすぐにかんじました」）：引述，表示她观察到的情况和感受。
     顔や足の形などを見て（かおやあしのかたちなどをみて）：动词，观察脸和脚的形状等。
     これはニホンオオカミだとすぐに感じました（これはにほんおおかみだとすぐにかんじました）：表示她立即感到这是日本狼。
   うれしくて、踊りたくなりました（うれしくて、おどりたくなりました）：描述她的感受，感到高兴，想要跳舞。
   と話していました（とはなしていました）：表示她正在说的过去式。
