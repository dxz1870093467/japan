《台湾の地震》
台湾で3日、大きな地震が起こりました。
5日の昼までに10人が亡くなって、1000人以上の人がけがをしました。大きな被害が出ています。
台湾の人たちは、今まで日本で大きな地震などが起こると、寄付のお金をたくさん送ってくれました。
2011年の東日本大震災のときは200億円以上、今年1月の能登半島地震のときは25億円以上の寄付をしてくれました。
このため台湾の地震のあと、SNSには「台湾が助けてくれたことを忘れません」とか「今度は日本が台湾を助けましょう」などのことばが多くなっています。
台湾と交流してきた市や町は、寄付のための箱を作ってお金を集めています。
コンビニや日本赤十字社、NGOなども寄付を集めています。

---

**台湾で3日、大きな地震が起こりました。**
- 读法：たいわん で みっか、 おおきな じしん が おこりました。
- 翻译：3号在台湾发生了大地震。
- 语法分析：
  - 「〜で」表示发生地点。
  - 「〜が起こりました」表示事情发生了。
  - 「大きな」是“大的”，形容词。

---

**5日の昼までに10人が亡くなって、1000人以上の人がけがをしました。大きな被害が出ています。**
- 读法：いつか の ひる まで に じゅうにん が なくなって、せんにん いじょう の ひと が けが を しました。 おおきな ひがい が でています。
- 翻译：到5日中午为止，已经有10人死亡，超过1000人受伤。发生了严重的灾害。
- 语法分析：
  - 「〜までに」表示截止时间。
  - 「亡くなって」是「亡くなる」的て形，表示前后动作连接。
  - 「〜以上」表示“以上”。
  - 「被害が出ています」表示“出现了灾害”，现在进行状态。

---

**台湾の人たちは、今まで日本で大きな地震などが起こると、寄付のお金をたくさん送ってくれました。**
- 读法：たいわん の ひとたち は、いままで にほん で おおきな じしん など が おこる と、きふ の おかね を たくさん おくってくれました。
- 翻译：台湾人以前在日本发生大地震等灾害时，送来了很多捐款。
- 语法分析：
  - 「〜など」表示“等等”，列举的意思。
  - 「〜てくれました」表示别人为说话人做了某事，带有感谢意味。

---

**2011年の東日本大震災のときは200億円以上、今年1月の能登半島地震のときは25億円以上の寄付をしてくれました。**
- 读法：にせんじゅういちねん の ひがしにほんだいしんさい の とき は にひゃくおくえん いじょう、ことし いちがつ の のとはんとうじしん の とき は にじゅうごおくえん いじょう の きふ を してくれました。
- 翻译：2011年东日本大地震时，捐款超过200亿日元，今年1月能登半岛地震时，捐款超过25亿日元。
- 语法分析：
  - 「〜のとき」表示时间点。
  - 「〜以上」表示数量以上。
  - 「寄付をしてくれました」表示别人主动做捐款。

---

**このため台湾の地震のあと、SNSには「台湾が助けてくれたことを忘れません」とか「今度は日本が台湾を助けましょう」などのことばが多くなっています。**
- 读法：このため たいわん の じしん の あと、エスエヌエス には 「たいわん が たすけてくれた こと を わすれません」 とか 「こんど は にほん が たいわん を たすけましょう」 など の ことば が おおく なっています。
- 翻译：因此，台湾地震发生后，在SNS上出现了许多“不会忘记台湾帮助我们的事”和“这次轮到日本帮助台湾了”等话语。
- 语法分析：
  - 「〜など」表示举例。
  - 「〜が多くなっています」表示数量变多，状态进行中。
  - 「〜とか」表示列举举例，类似“比如”。

---

**台湾と交流してきた市や町は、寄付のための箱を作ってお金を集めています。**
- 读法：たいわん と こうりゅう してきた し や まち は、きふ の ため の はこ を つくって おかね を あつめています。
- 翻译：与台湾有交流的市镇，制作了募捐箱正在募集资金。
- 语法分析：
  - 「〜してきた」表示从过去到现在的动作持续。
  - 「〜ための」表示目的或用途。
  - 「〜ています」表示正在进行。

---

**コンビニや日本赤十字社、NGOなども寄付を集めています。**
- 读法：コンビニ や にほんせきじゅうじしゃ、エヌジーオー など も きふ を あつめています。
- 翻译：便利店、日本红十字会、NGO等也在募集捐款。
- 语法分析：
  - 「〜や〜など」表示列举。
  - 「〜も」表示“也”。
  - 「〜ています」表示正在进行。
