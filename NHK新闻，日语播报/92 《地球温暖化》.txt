《地球温暖化》
3年前、国連のUNDPがビデオを作りました。
ビデオでは、恐竜のキャラクターのフランキーが「恐竜は、地球に落ちた大きな石で絶滅しました。
人は、このままでは地球の気温を上げ続けて、自分たちで絶滅してしまいます」と言いました。
フランキーは今まで、アメリカやエジプトなどに行ってメッセージを伝えました。今月、初めて日本に来ました。
8日、フランキーは東京の渋谷で「みんなが一緒に頑張ったら、地球を守ることができます」と話しました。
そして、若い人たちと一緒にまちを歩きながら「地球の気温を上げるのを止めるチャンスは今だけです」と言いました。
フランキーは13日まで、日本でイベントに出たり、国会議員に会ったりする予定です。





 １．

原文：
3年前、国連のUNDPがビデオを作りました。

读法：
さんねんまえ、こくれん の UNDP が ビデオ を つくりました。

翻译：
三年前，联合国开发计划署（UNDP）制作了一个视频。

语句分析：

 「3年前」是时间状语。
 「国連のUNDP」是主语，指联合国开发计划署。
 「ビデオを作りました」是谓语，表示动作“制作了视频”。



 ２．

原文：
ビデオでは、恐竜のキャラクターのフランキーが「恐竜は、地球に落ちた大きな石で絶滅しました。

读法：
ビデオ で は、きょうりゅう の キャラクター の フランキー が 「きょうりゅう は、ちきゅう に おちた おおきな いし で ぜつめつしました。

翻译：
视频中，恐龙的角色弗兰基说：“恐龙是因为撞击地球的大石头而灭绝的。

语句分析：

 「ビデオでは」表明场所、对象。
 「恐竜のキャラクターのフランキー」是主语。
 引号内为引用的直接话语。
 「地球に落ちた大きな石で絶滅しました」解释原因和结果。



 ３．

原文：
人は、このままでは地球の気温を上げ続けて、自分たちで絶滅してしまいます」と言いました。

读法：
ひと は、このまま で は ちきゅう の きおん を あげつづけて、じぶんたち で ぜつめつ してしまいます」と いいました。

翻译：
他说：“人类如果照这样继续下去，将持续提升地球气温，最终自我灭绝。”

语句分析：

 「このままでは」表示假设条件，“照这样下去”。
 「地球の気温を上げ続けて」表示持续动作。
 「自分たちで絶滅してしまいます」说明结果，带有不可避免的感觉。
 「と言いました」表示说话动作。



 ４．

原文：
フランキーは今まで、アメリカやエジプトなどに行ってメッセージを伝えました。今月、初めて日本に来ました。

读法：
フランキー は いままで、アメリカ や エジプト など に いって メッセージ を つたえました。こんげつ、はじめて にほん に きました。

翻译：
弗兰基至今曾去过美国、埃及等地传达信息。本月首次来到日本。

语句分析：

 「今まで」表示时间范围“直到现在”。
 「アメリカやエジプトなどに行って」列举地点，表示去过的地方。
 「メッセージを伝えました」表示传达信息的动作。
 「今月、初めて日本に来ました」表示时间点及动作。



 ５．

原文：
8日、フランキーは東京の渋谷で「みんなが一緒に頑張ったら、地球を守ることができます」と話しました。

读法：
ようか、フランキー は とうきょう の しぶや で 「みんな が いっしょ に がんばったら、ちきゅう を まもる こと が できます」と はなしました。

翻译：
8日，弗兰基在东京涩谷说：“大家一起努力的话，就能保护地球。”

语句分析：

 「8日」为时间。
 「フランキーは東京の渋谷で話しました」是主句，说明人物、地点和动作。
 引号内为直接引语，表示条件与结果。



 ６．

原文：
そして、若い人たちと一緒にまちを歩きながら「地球の気温を上げるのを止めるチャンスは今だけです」と言いました。

读法：
そして、わかい ひとたち と いっしょ に まち を あるきながら 「ちきゅう の きおん を あげる の を とめる チャンス は いま だけ です」と いいました。

翻译：
然后，他与年轻人一起走在街上，说：“阻止地球气温上升的机会只有现在。”

语句分析：

 「若い人たちと一緒にまちを歩きながら」表示同时进行的动作。
 引号内为引语，说明重要的观点和时机。
 「と言いました」引出说话内容。



 ７．

原文：
フランキーは13日まで、日本でイベントに出たり、国会議員に会ったりする予定です。

读法：
フランキー は じゅうさんにち まで、にほん で イベント に でたり、こっかいぎいん に あったり する よてい です。

翻译：
弗兰基计划在13日之前，在日本参加活动，并会见国会议员等。

语句分析：

 「13日まで」表示期限。
 「イベントに出たり、国会議員に会ったりする」使用了“〜たり〜たり”的列举形式，表示做多项活动。
 「予定です」表示计划。
