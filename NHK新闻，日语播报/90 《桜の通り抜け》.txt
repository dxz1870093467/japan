《桜の通り抜け》
大阪の造幣局の中には、340本の桜が並ぶ560mぐらいの道があります。141種類の桜の花が咲きます。
造幣局は、桜の花が咲くころ、みんながこの道を通ることができるようにする「桜の通り抜け」を行っています。
140年以上前から続いていて、今年は5日に始まりました。
造幣局は毎年「今年の花」を選んでいます。
今年は「大手毬（おおてまり）」という桜で、薄い紅色の花が枝の先に集まって咲く、きれいな花です。
見に来た人は「きれいな桜と一緒に赤ちゃんの写真を撮ることができて、うれしかったです」と話していました。
「桜の通り抜け」は11日までで、見に行く人はインターネットで予約が必要です。


**大阪の造幣局の中には、340本の桜が並ぶ560mぐらいの道があります。**
- 读法：おおさか の ぞうへいきょく の なか に は、さんびゃくよんじゅう ほん の さくら が ならぶ ごひゃくろくじゅうメートル ぐらい の みち が あります。
- 释义：大阪的造币局内，有一条大约560米长、种有340棵樱花树的道路。
- 语法重点：
  - 「〜の中には」表示“……里面有……”。
  - 「並ぶ」动词，表示“排列，排列着”。
  - 「ぐらい」表示“大约”。

---

**141種類の桜の花が咲きます。**
- 读法：ひゃくよんじゅういっしゅるい の さくら の はな が さきます。
- 释义：会开出141种类的樱花。
- 语法重点：
  - 「種類」表示“种类”。
  - 「咲きます」表示“开花”。

---

**造幣局は、桜の花が咲くころ、みんながこの道を通ることができるようにする「桜の通り抜け」を行っています。**
- 读法：ぞうへいきょく は、さくら の はな が さく ころ、みんな が この みち を とおる こと が できる よう に する 「さくら の とおりぬけ」 を おこなっています。
- 释义：造币局在樱花盛开的时候，会举办“樱花通行”活动，让大家可以走过这条路。
- 语法重点：
  - 「〜ころ」表示“……的时候”。
  - 「〜ようにする」表示“让……能够……”。
  - 「行っています」表示“正在举办，进行”。

---

**140年以上前から続いていて、今年は5日に始まりました。**
- 读法：ひゃくよんじゅう ねん いじょう まえ から つづいていて、ことし は いつか に はじまりました。
- 释义：这项活动从140多年前开始持续举办，今年在5号开始。
- 语法重点：
  - 「〜から続いていて」表示“从……开始持续”。
  - 「始まりました」表示“开始了”。

---

**造幣局は毎年「今年の花」を選んでいます。**
- 读法：ぞうへいきょく は まいとし 「ことし の はな」 を えらんでいます。
- 释义：造币局每年都会选出“今年的花”。
- 语法重点：
  - 「毎年」表示“每年”。
  - 「選んでいます」表示“正在选择（习惯性动作）”。

---

**今年は「大手毬（おおてまり）」という桜で、薄い紅色の花が枝の先に集まって咲く、きれいな花です。**
- 读法：ことし は 「おおてまり」 という さくら で、うすい べにいろ の はな が えだ の さき に あつまって さく、きれい な はな です。
- 释义：今年选的是叫“大手毬”的樱花，花色为淡淡的红色，花朵聚集在枝头，十分漂亮。
- 语法重点：
  - 「〜という」表示“叫做……”（介绍名称）。
  - 「〜が〜に集まって咲く」表示“……聚集在……上开花”。

---

**見に来た人は「きれいな桜と一緒に赤ちゃんの写真を撮ることができて、うれしかったです」と話していました。**
- 读法：みにきた ひと は 「きれいな さくら と いっしょ に あかちゃん の しゃしん を とる こと が できて、うれしかった です」 と はなしていました。
- 释义：来看花的人说：“能和漂亮的樱花一起给宝宝拍照，感到很开心。”
- 语法重点：
  - 「〜ことができて」表示“能够……”。
  - 「〜と話していました」表示“说（过去的事情）”。

---

**「桜の通り抜け」は11日までで、見に行く人はインターネットで予約が必要です。**
- 读法：「さくら の とおりぬけ」 は じゅういちにち まで で、みにいく ひと は インターネット で よやく が ひつよう です。
- 释义：“樱花通行”活动持续到11号，想去参观的人需要通过互联网预约。
- 语法重点：
  - 「〜までで」表示“持续到……为止”。
  - 「〜が必要です」表示“需要……”。
