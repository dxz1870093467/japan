#### 1. 自发动词（自动词）与他动词的对应

| 自发动词（自动词） | 他动词 | 备注 |
|---|---|---|
| 閉まります（门、窗）| 閉めます | 关、关闭 |
| 壊れます | 壊します | 破坏、弄坏 |
| 割れます | 割ります | 打破 |
| 付きます（附加）| 付けます | 附加、贴上 |
| つきます（电器、灯）| つけます | 打开（灯、设备） |
| かかります（挂、花费）| かけます | 挂上、花费 |
| 汚れます | 汚します | 弄脏 |
| 止まります（风、雨）| × | 无对应他动词 |
| 止まります（机器、车）| 止めます／とめます／とめます | 停止 |
| 建ちます | 建てます | 建造 |
| 並びます | 並べます | 排列 |
| 消えます（灯、电器）| 消します | 关掉、消除 |
| 落ちます | 落とします | 掉落 |

（其它如：被る（帽子）、履く（下半身穿的）、着る（上半身穿的）、飼う（养动物）等穿戴或实物用法，根据具体词汇区分）

#### 2. 「～ている」的用法

- 表示正在进行
- 表示动作结果保持（状态）
- 表示动作习惯（职业、习惯）

- 【本课重点】  
  动词基本形 + ている  
  表示结果状态或动作的持续。（物体现在的状态）
- 自动词的た形也可以表示结果存续（同ている用法）

- 例句：
    - 部屋の電気が消えてしまいます。（房间的灯灭了。）
    - 壁に絵が掛かっています。（墙上挂着画。）

---

#### 3. 「～てしまいます」的用法

- 简体口语形：
    - ～てしまう → ちゃう（口语）
    - ～でしまう → じゃう（口语）
    - 过去式：～てしまった、～でしまった → ちゃった、じゃった

- 用法：
    1. 表示动作全部完成（彻底完成）
    2. 表示造成无法挽回的结果，有遗憾、懊悔或责备的心情。

- 例句：
    - 泣いてしまいました。（哭了。）
    - 読んでしまいました。（读完了。）
    - このほんはもう読んでしまいました。（这本书已经读了。）
    - 財布をうっかり忘れてしまいました。（不小心忘带钱包了。）

---

#### 4. 「～そうです」——样态推测（根据外观、第一印象推测）

- 结构：
    - 动词去ます + そうです
    - い形容词去い + そうです
    - な形容词词干 + そうです

- 特殊变化：
    - い形容词「いい」→「よさそう」
    - 否定形：
        - おいしい → おいしそう（好吃的样子）
        - おいしくない → おいしくなさそう（看起来不好吃）

- 例句：
    - このケーキはおいしそうです。（这个蛋糕看起来很好吃。）
    - 雨が降りそうです。（好像要下雨了。）
    - 面白そうな本を買いました。（买了一本看起来有趣的书。）
    - 楽しそうに遊んでいる。（玩得很开心。）
    - 今日は会議がなさそうです。（今天好像没有会议。）
    - 雨が止みそうもない。（看起来雨不会停。）
