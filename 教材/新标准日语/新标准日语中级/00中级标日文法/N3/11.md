# 第11课 语法笔记整理

---

## 1. ～わけではない / ～わけでもない
- **意思**：并非……，不一定是……（否定某种极端或绝对的判断）
- **构成**：
    - 名词 + である
    - 形2 + わけではない
    - 形1 动简　+ わけではない

- **用法说明**：用来纠正或缓和前述的判断，表示“并不是那样”的意思。
- **例句**（句子简体+）：  
  彼の気持ちは理解できないわけではないが、そのやり方には賛成できない。  
  （并不是不能理解他的感情，但不能赞成他的做法。）

---

## 2. ～にしても/にしたって（口语）／にしろ／にせよ
- **意思**：无论……还是……，不论……都……（表示让步）
- **构成**：
    - 名词/形2 + にしても / にしろ / にせよ
    - 形1/动词简体 + にしても / にしろ / にせよ

- **说明**：口语中常说「～にしたって」，是「～にしても」的口语形式。
- **相似句型**：～ても～ても，同样表达“无论……还是……”
- **例句**（形1/动词简体）：  
  同一类别：肉にしても魚にしても、新鮮なものはおいしいです。  
  （无论是肉还是鱼，新鲜的都好吃。）

  相反：行かないにしろ、前もって電話してください。  
  （即使不去，也请提前打电话。）

## 3. ～向けだ / ～向けの+名词 / ～向けに+动词（面向...）
- **意思**：面向……，针对……（表示对象或用途）
- **构成**：
    - 名词 + 向けだ / 向けの+名词 / 向けに+动词（面向...）
    - 名词 + 向きだ / 向きの+名词 / 向きに+动词（适合...）

- **例句**：  
  若者向けに服をデザインした。  
  （为年轻人设计了衣服。）

  畳はやっぱり日本人向きだ。  
  （榻榻米果然适合日本人。）

  名词 + に向いている → 适合……

## 4. ～だけでなく ～も （不仅...而且...）/ ～ばかりか ～も /（うえに 好+好/坏+坏）
- **意思**：不仅……而且……（表示递进）
- **构成**：


- **例句**：  
  このカメラは小型であるうえに、使いやすい。  
  （这台相机不仅小巧，而且好用。）

  あいつは仲間を裏切ったうえ、お金を持って逃げた。  
  （那家伙不仅背叛了伙伴，还带着钱逃跑了。）

## 5. ～とともに（～と共に）
- **意思**：与……一起，随着……
- **构成**：
    - 名词 + とともに
    - 动词辞书形 + とともに

- **用法**：
    1. 和“といっしょに”一样  和...一起...
    2. ...同时（と　同時に） 
    3. 随着...

- **2.例句**：  
  彼は作家とともに教師でもある。  
  （他既是作家，也是教师。）
- **3.例句**：
  年を取るとともに、体が弱ってきたようだ。  
  （随着年龄增长，身体也变弱了。）

- **3.相关搭配**：
    - に従って（前项faf）
    - に伴って（随着）
    - につれて（随着）
