# 第24课 语法笔记整理

---

## 1. ～がちだ
- **构成**：
    - 名词 + がち
    - 动词ます形去ます + がち
- **意思**：倾向于……，常常……（表示某种不良倾向或负面状态）
- **例句**：
    - 最近、曇りがちの天気が続いた。  
      （最近，阴天的天气持续了。）
    - 病気がちになる。  
      （容易生病。）

---

## 2. ～以上（は） / ～上は
- **构成**：名词 + である + 以上（は） / 上は  
  形容词 / 动词简体 + 以上（は） / 上は
- **意思**：既然……就……（表示既定事实或条件，后一部分表达必然的结果或义务）
- **例句**：
    - 学生である以上、ルールを守らなければならない。  
      （既然是学生，就必须遵守规章。）
    - 脱然学生、最必遵守规定。

---

## 3. ～がる
- **构成**：形容词 + がる（动词）
- **意思**：表示第三人称的感情或愿望，表达“想要……、觉得……”
- **说明**：用于表别人（第三人称）的心理状态，不能用第一人称
- **例句**：
    - ～がほしい → ～をほしがる（第三人称想要）
    - ～たい → ～たがる
- **例句**：
    - 子供がジュースを飲みたがります。  
      （孩子想喝果汁。）
- **特殊**：
    - 形容词“嫌い”变为“嫌がる”表示“觉得讨厌”或“害怕”（带有“いや音”）

---

## 4. ～わけだ
- **构成**：
    - 名词 + である / な + わけだ
    - 形容词 + な + わけだ
    - 形容动词 / 动词简体 + わけだ
- **意思**：
    1) 理所当然，是……的理由（表示原因或理所当然的结果）
    2) 也就是说（解释理由或结论）
- **例句**：
    1) 38度もある。暑いわけだ。  
       （有38度，难怪这么热。）
    2) 入場料は一人200円だから、5人は1000円というわけだ。  
       （入场费一个人200日元，5个人就是1000日元了。）
- **相关表达**：
    - わけにはいかない（不能……）
    - わけがない（不可能……）
    - わけではない（并非……）
