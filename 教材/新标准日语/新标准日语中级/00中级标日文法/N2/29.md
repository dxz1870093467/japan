# 第29课 语法笔记整理

---

## 1. ～必要はない / ～まででもない / ～ことはない
- **用法说明**：  
  表示“不必要……”，“不必……”
- **结构**：  
  动词基本形 + 必要はない  
  动词基本形 + まででもない  
  动词基本形 + ことはない
- **例句**：
    - 簡単だから、詳しく説明するまでもない。  
      （因为很简单，不必详细说明。）

---

## 2. ～ものの / ～とはいうものの
- **用法说明**：  
  表示“虽然……但是……”（转折）
- **结构**：  
  名词 + である + ものの  
  形容动词 + である + ものの  
  形容词 / 动词简体形 + ものの
- **例句**：
    - その本を読んだことがあるものの、あまり覚えていない。  
      （虽然读过那本书，但不太记得了。）
    - 雨とはいうものの、外に出かけた。  
      （虽然说下雨了，但还是出门了。）

---

## 3. ～にあたって / ～にあたり
- **用法说明**：  
  表示“在……之际”，“值此……时” （正式场合用语）
- **结构**：  
  名词 + にあたって / にあたり  
  动词基本形 + にあたって（较少用）
- **例句**：
    - 就任にあたって、一言ごあいさついたします。  
      （就职之际，致以简短的问候。）
    - 就任にあたり、我々一同頑張ります。  
      （值此就职之际，我们大家一起努力。）
