# 第31课 语法笔记整理

---

## 1. ～くせに / ～くせして
- **用法说明**：  
  表示“明明……却……”带有抱怨、指责、不满的语气，情感更加强烈。
- **结构**：  
  名词 + の + くせに  
  形容词 + くせに  
  动词简体形 + くせに
- **例句**：
    - 夫は一日ずっと家にいたくせに、掃除も洗濯もまったくやってない。  
      （丈夫明明整天呆在家里，完全没有做打扫和洗衣服。）
    - 毎日掃除するくせに、部屋はきれいじゃない。  
      （明明每天打扫，房间却不干净。）

---

## 2. ～べきだ
- **用法说明**：  
  表示“应该……”，用于义务或责任。
- **结构**：  
  形容词去い + く + なる + べきだ  
  名词、形容动词 + である + べきだ  
  动词基本形 + べきだ
- **例句**：
    - 警察は人の安全を守るべきだ。  
      （警察应该保护人们的安全。）
- **反义表达**：～べきではない（不应该）
    - 教室で大きな声で騒ぐべきではない。  
      （不应该在教室大声喧哗。）

---

## 3. ～もんか / ～ものか / ～もんですか
- **用法说明**：  
  强烈否定，表示“绝不……”
- **结构**：  
  名词、形容动词 + な + もんか  
  形容词、动词简体形 + もんか
- **例句**：
    - こんなサービスが悪い店には二度と行くものか。  
      （绝不会再去这种服务差的店。）

---

## 4. ～となると / ～となれば / ～とすれば / ～としたら
- **用法说明**：  
  表示“假如……”，“一旦……就……”
- **结构**：  
  动词简体形  
  名词、形容动词、形容词 + となると / となれば / とすれば / としたら
- **例句**：  
  ① 假如……
    - 海外で運転するとなると、その国の交通規則を知らなければならない。  
      （假如在海外开车，就必须了解那个国家的交通规则。）  
      ② 一旦……就……
    - 何度も練習したが、本番となると緊張してしまう。  
      （虽然练习了很多次，一旦到了正式场合就会紧张。）
