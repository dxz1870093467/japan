# 第27课 语法笔记整理

---

## 1. ～なしには / ～なしでは（+否定）
- **用法说明**：  
  表示“没有……就不能……”
- **结构**：  
  名词 + なしには / なしでは  
  动词た形 + こと + なしには
- **例句**：
    - 調査することなしには、発言できない。  
      （没有调查就不能发表意见。）

---

## 2. 动词ます形 + 得る / 得る（うる・える）（有可能）
- **用法说明**：  
  表示“有可能发生……”
- **例句**：
    - 今から頑張ると、志望の大学に入ることも十分ありうる。  
      （现在努力的话，有可能进入理想的大学。）
- **否定形 + 得ない（えない）**：表示“没有可能，不可能”
- **例句**：
    - 一人では生きていくことがありえない。  
      （一个人是不可能活下去的。）

---

## 3. ～をもとにして / ～をもとにする + 名词（根据、依据）
- **用法说明**：  
  表示“以……为依据，根据……”
- **例句**：
    - 調査結果をもとにして、今後の方針を決める。  
      （根据调查结果决定今后的方针。）

---

## 4. まじ（＝～まい）
- **用法说明**：  
  用于表示否定推测、否定意志、或者劝诱、意志的否定形式
- **结构**：  
  动词 + まじ  
  动词 + まい / するまい / すまい / しまい / くるまい / こまい
- **用法解析**：  
  ① 否定推测（=～ないだろう / ～ないでしょう）
    - もう春だから雪が降るまじ。  
      （已经是春天了，不会下雪。）  
      ② 否定意志（=～しないつもり）
    - あんな所二度と行くまじ。  
      （打算再也不去那种地方。）  
      ③ ～ようか～まいか（要不要……，犹豫）
    - 行こうか行くまいか後で決める。  
      （要不要去，待会儿决定。）
- **附注**：
    - ～ではあるまいか / ～ではまいか = 不是……吗？
    - ～ではありませんか（更礼貌的说法）

---

## 5. ～ずにはいられない / ～ないではいられない（禁不住、不得不）
- **用法说明**：  
  表示“不得不……，禁不住……”
- **结构**：  
  动词否定形 + ずにはいられない
- **例句**：
    - 人間は呼吸せずにはいられない。  
      （人不能不呼吸。）

---

## 6. ～のみならず（不仅……而且……）
- **用法说明**：  
  表示不仅一方，而且另一方也……，强调递进
- **相关表达**（类似用法）：  
  ① ～ばかりか  
  ② ～ばかりでなく  
  ③ ～だけでなく  
  ④ ～のみか  
  ⑤ ～のみならず  
  ⑥ ～に限らず  
  ⑦ ～上に  
  ⑧ ～に加えて
