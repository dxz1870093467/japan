# 第32课 语法笔记整理

---

## 1. ～てしかたがない / ～てしょうがない / ～てしかたがない / ～てたまらない / ～てならない / ～てかなわない
- **用法说明**：  
  表示“非常……”“不得了”，表达强烈的感情或状态。
- **结构**：  
  动词て形 / 形容词て形 + しかたがない / しょうがない / たまらない / ならない / かなわない
- **例句**：
    - プレゼントをもらって、うれしくてたまらない。  
      （收到礼物，非常开心，忍不住高兴。）

---

## 2. ～どころではない
- **用法说明**：  
  表示“没有……的余裕/条件”，指当前情况不允许做某事。
- **结构**：  
  名词 + どころではない  
  动词简体形 + どころではない
- **例句**：
    - 家を買ったばかりで、お金がないので海外旅行に行くどころではない。  
      （刚买了房子，没有钱去海外旅行。）

---

## 3. ～なり / ～なりの / ～なりに
- **用法说明**：  
  表示“一种相应的状态、能力或行为方式”，强调“按照……自己的方式”
- **结构**：  
  名词 + なり / なりの  
  形容动词、形容词、动词简体形 + なりに
- **例句**：
    - 私は私なりに人生観を持っている。  
      （我有我自己的人生观。）
    - 子供には子供なりの楽しみがある。  
      （孩子有孩子自己喜欢的娱乐方式。）

---

## 4. ～でしかない / ～にすぎない
- **用法说明**：  
  表示“不过是……”，强调限定范围（有强调语气的作用）。
- **结构**：  
  名词 + でしかない
- **例句**：
    - 台湾はもちろん中国の一部分でしかない。  
      （台湾无疑是中国的一部分。）

---

## 5. ～おそれがある / ～おそれもある
- **用法说明**：  
  表示“有可能……，恐怕会……”（有负面含义）
- **结构**：  
  名词 + の + おそれがある  
  动词简体形 + おそれがある
- **例句**：
    - 地震のために、この地域は断水のおそれがある。  
      （因为地震，这个地区可能会断水。）

---

## 6. ～に至る
- **用法说明**：  
  表示“达到、到达（时间、地点、程度）”
- **结构**：  
  名词 + に至る  
  动词连用形 + に至る
- **例句**：
    - 11時午後北京に至る。  
      （下午11点到达北京。）
    - 話し合いは深夜に至る。  
      （讨论直到深夜。）

---

## 7. ～かたわら / ～かたわら
- **用法说明**：  
  表示“一边……一边……”（主语相同，两个动作同时进行）
- **结构**：  
  名词 + の + かたわら  
  动词基本形 + かたわら
- **例句**：
    - 彼は銀行員として働くかたわら、小説も書きます。  
      （他一边当银行职员，一边写小说。）

---

## 8. ～をもって
- **用法说明**：  
  ① 表示时间的起点或结束（书面语）  
  ② 表示手段、方式（较正式）
- **结构**：  
  名词（时间） + をもって  
  名词（手段、方式） + をもって
- **例句**：  
  ① 当店は7時をもって閉店させていただきます。  
  （本店于7点关闭。）  
  ② 書面をもってお知らせいたします。  
  （以书面形式通知。）
