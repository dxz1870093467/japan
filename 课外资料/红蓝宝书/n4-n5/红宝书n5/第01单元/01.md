# 第1单元 词汇整理

1. **あ（嗄）**
    - 词性：感叹词
    - 意思：突然想起或感动时发出的声音
    - 例句：あ、お金がなくなった。/ 哎，钱没了。
      あ、もう時間だ。/ 哎，快到时间了。

2. **ああ（嗄）**
    - 词性：感叹词
    - 意思：表示感谢、惊奇、叹息等感情
    - 例句：ああ、ありがとうございます。/ 啊，谢谢。
      ああ、驚きました。/ 啊，真惊讶。

3. **アイスクリーム（ice cream）**
    - 词性：名词
    - 意思：冰淇淋
    - 相关词：アイスクリームケーキ（冰淇淋蛋糕）
      アイス（ice）（冰淇淋）

4. **間（あいだ）**
    - 词性：名词
    - 意思：中间，之间，关系
    - 例句：便局と本屋の間 / 邮局和书店之间
      クラスメートの間 / 同班同学之间

5. **仲（なか）**
    - 词性：名词
    - 意思：内部，中间；关系
    - 释义：人际关系等

6. **相手（あいて）**
    - 词性：名词
    - 意思：对象，对手
    - 例句：話し相手 / 谈话对象
      相手のチーム / 对方队伍

7. **IDカード（アイディカード）**
    - 词性：名词
    - 意思：身份证
    - 例句：IDカードを落とした。/ 丢了身份证。
      私に見せる。/ 给我看一下你的身份证。

8. **身分証明書（みぶんしょうめいしょ）**

9. **IT産業（アイティーさんぎょう）**

    - 例句：IT産業の会社 / IT产业公司
      IT産業の進む方向 / IT产业的发展方向

10. **会う（あう）**
