# JLPT N2 动词变形学习重点和内容

---
## 一、基本情况
N2是中高级水平，强调日常和商务场合中较为复杂的表达能力。动词变形不仅要熟练掌握各种语法形态，还要能够灵活运用各种敬语、语气变化和复合句型。
---
## 二、N2重点掌握的动词变形类别

| 变形类别       | 内容说明                                                        | 例子（动词「書く」）                  |
|----------------|-----------------------------------------------------------------|------------------------------------|
| 1. 各种敬语形态 | 尊敬语、谦让语、礼貌语的变形及用法                              | お書きになる（尊敬语）、書かせていただく（谦让语） |
| 2. 使役形、使役被动形 | 表示“让别人做”、“被迫做”的复杂表达                                     | 書かせる、書かせられる                 |
| 3. 意志形、条件形、假定形 | 各种条件句假设句型的多样用法                                      | 書こう、書けば、書いたら                     |
| 4. て形的多样用法 | て形连接复句、表达原因、顺接、并列、让步等多样表达                         | 書いて、書いても                      |
| 5. 可能形、被动态   | 能力、可能性与被动表达的更加细致使用                                | 書ける、書かれる                       |
| 6. 各类复合动词、自动词与他动词 | 区分自动词和他动词活用，常用复合动词动词的掌握                           | 落ちる（自动），落とす（他动）          |
| 7. 意愿、推测、命令等语气表达 | 多样的表达愿望、推测、要求的动词活用                                   | 書きたい、書くだろう、書け                 |
| 8. 各类动词的连用形与连体形的细致区分 | 用于连接名词、形容词、助词等的多样组合                                   | 書く+もの、書いた+ところ               |
| 9. 动词与助词结合的特殊表达 | 例如「〜てほしい」「〜てみる」「〜ておく」等复杂表达                     | 書いてほしい、書いてみる、書いておく       |

## 三、具体变形介绍

### 1. 尊敬语 / 谦让语动词活用举例

- お〜になる / 〜られる
  例：社長がお書きになる（尊敬）
- 〜させていただく（谦让）
  例：お手伝いさせていただきます

### 2. 使役形、使役被动形

| 变形     | 说明                         | 例子               |
|----------|------------------------------|--------------------|
| 使役形   | 让别人做某事                   | 書かせる（让写）     |
| 使役被动形 | 被迫做某事                     | 書かせられる（被迫写）|

### 3. 条件形、假定形多样用法

| 变形     | 用法说明                         | 例子                   |
|----------|---------------------------------|------------------------|
| ば形     | 如果…就…，常用条件表达           | 書けば                   |
| たら形   | 表示条件或时间点                 | 書いたら                  |
| と形     | 表示自然结果或习惯性动作           | 書くと…                  |

### 4. て形多用法

| 用途         | 说明                        | 例句                     |
|--------------|-----------------------------|--------------------------|
| 连接动作     | 顺接动作                    | 書いて、話して、帰る        |
| 表示原因、理由 | 表示原因或理由               | 熱があって、学校に行けない   |
| 让步         | 尽管…，但是                 | 雨が降っても、行きます      |
| 请求         | 请求别人做某事               | 書いてください            |

### 5. 可能形与被动态

| 变形     | 说明                  | 例子             |
|----------|-----------------------|------------------|
| 可能形   | 能做…                  | 書ける           |
| 被动形   | 被……做                 | 書かれる         |

## 四、动词语法知识点扩展（N2常见）

| 语法点                         | 说明                              | 例子                         |
|--------------------------------|---------------------------------|------------------------------|
| 〜ている vs 〜てある            | 进行状态 vs 结果状态              | 書いている（正在写） vs 書いてある（写好了） |
| 〜てしまう                     | 表示动作完成或遗憾                | 書いてしまった                |
| 〜ておく                       | 事先做某事                      | 書いておく                    |
| 〜てみる                       | 试着做某事                      | 書いてみる                    |
| 〜ながら                       | 表示同时进行                    | 書きながら話す                |
| 〜させる / 〜させられる        | 使役与使役被动态                  | 書かせる / 書かせられる        |
| 〜たい                         | 表示愿望                        | 書きたい                    |
| 〜ようと思う                   | 表示意图                        | 書こうと思う                |

---

## 五、学习建议

- **系统复习N3已学动词变形**，确保熟练。
- **重点掌握使役形、使役被动态以及复杂敬语表达**，因为商务和正式场合常用。
- **练习多种条件句形（ば形、たら形、と形）的不同用法和细微差异。**
- **多做综合句型练习，尤其是用て形连接的各种表达。**
- **扩展词汇，尤其是自动词和他动词的区分及使用场景。**
